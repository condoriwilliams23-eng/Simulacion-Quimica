===========================================================================
     PROYECTO DE INVESTIGACIÓN: SIMULADOR DE PRECIPITADO Y FILTRACIÓN
===========================================================================

ESTADO: Proyecto de Beca de Investigación
PLATAFORMA DE DESARROLLO: Construct (Software de Simulación Interactiva)
INSTITUCIÓN: Universidad Tecnológica Nacional - Facultad Regional Santa Fe

---------------------------------------------------------------------------
1. DESCRIPCIÓN DEL PROYECTO
---------------------------------------------------------------------------
Este simulador ha sido desarrollado como parte de un proyecto de beca de 
investigación con el objetivo de modelar visual y funcionalmente los 
procesos industriales de precipitación y filtración. 

La herramienta permite a los usuarios interactuar con las variables del 
proceso en un entorno controlado, facilitando la comprensión de fenómenos 
químicos y mecánicos que ocurren en la separación de mezclas.

---------------------------------------------------------------------------
2. ESTRUCTURA DEL SOFTWARE
---------------------------------------------------------------------------
El proyecto se organiza en diferentes "Layouts" o escenas interactivas:

- INICIO: Pantalla de bienvenida y configuración inicial de la simulación.
- ÍNDICE: Navegación centralizada entre las diferentes etapas del proceso.
- SIMULADOR: El núcleo del proyecto, donde se ejecutan los cálculos y se 
  visualiza el comportamiento del precipitado y la eficiencia de filtración.

---------------------------------------------------------------------------
3. COMPONENTES TÉCNICOS (Lógica del Modelo)
---------------------------------------------------------------------------
El simulador utiliza una arquitectura basada en eventos para gestionar:
- Dinámica de partículas: Representación visual del precipitado.
- Control de estados: Gestión de las transiciones entre las etapas de 
  mezcla, reacción y filtrado.
- Interfaz de Usuario (UI): Visualización en tiempo real de los datos del 
  proceso mediante elementos XML de estado.

---------------------------------------------------------------------------
4. REQUISITOS Y EJECUCIÓN
---------------------------------------------------------------------------
Para visualizar o editar el proyecto, se requiere:
- Construct 2 o Construct 3.
- Abrir el archivo principal: "Simulador precipitado y filtracion.caproj".

Los archivos complementarios (.uistate.xml y .xml) contienen la lógica de 
posicionamiento y las variables necesarias para el correcto funcionamiento 
de los objetos en pantalla.

---------------------------------------------------------------------------
5. OBJETIVOS DE INVESTIGACIÓN ALCANZADOS
---------------------------------------------------------------------------
- Modelado interactivo de procesos físico-químicos.
- Creación de una interfaz educativa para la transferencia de conocimiento.
- Análisis visual del flujo de filtración bajo diferentes parámetros.
